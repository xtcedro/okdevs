Declaración de Accesibilidad

Nuestro compromiso de garantizar que nuestro sitio web sea accesible para todos.

Fecha de vigencia: 1 de enero de 2025

1. Nuestro Compromiso con la Accesibilidad

En OKDEVS, estamos comprometidos a proporcionar un sitio web accesible para todos, independientemente de sus capacidades o discapacidades. Nuestro objetivo es garantizar que todos los usuarios tengan una experiencia fluida al navegar por nuestro sitio web y acceder a nuestro contenido.

2. Estándares de Accesibilidad

Este sitio web ha sido diseñado para cumplir con las Pautas de Accesibilidad al Contenido en la Web (WCAG) 2.1, Nivel AA. Estas pautas ayudan a garantizar que el contenido sea accesible para una amplia gama de personas con discapacidades, incluidas aquellas con ceguera, baja visión, pérdida auditiva y discapacidades cognitivas.

3. Características de Nuestro Sitio Web Accesible

  • Alternativas de texto para contenido no textual.
  • Navegación por teclado para todos los elementos interactivos.
  • Relaciones de contraste que hacen que el texto y los elementos visuales sean legibles.
  • Diseño responsivo que se adapta a diferentes tamaños de pantalla y tecnologías de asistencia.

4. Mejoras Continuas

Continuamente estamos trabajando para mejorar la accesibilidad de nuestro sitio web. Los comentarios de nuestros usuarios son vitales para ayudarnos a lograr nuestros objetivos.

5. Soporte de Accesibilidad

Si encuentra algún problema de accesibilidad al usar nuestro sitio web, por favor contáctenos. Estamos aquí para ayudar y atender cualquier inquietud que pueda tener.
